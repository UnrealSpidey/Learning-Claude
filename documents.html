Overview

What You Will Learn

This lesson covers Claude's capabilities for working with files and documents — reading spreadsheets, parsing PDFs, comparing documents, extracting structured data, and generating professional deliverables. You will learn when to use each skill and how to chain them for complex document workflows.

Working With PDFs

What Claude Can Do With PDFs

  • Extract all text, preserving structure and headings
  • Pull specific tables as structured data
  • Summarise long documents with configurable depth
  • Compare two PDF versions and highlight differences
  • Merge multiple PDFs into one
  • Fill in form fields (with your approval)
  • Create new PDFs with custom layouts and branding

Extracting Structured Data From PDFs

  • 'Read this invoice PDF and extract: vendor name, date, line items, and total. Save as a spreadsheet.'
  • 'This PDF contains a data table on pages 3–5. Extract it as a CSV.'
  • 'Read all five of these PDFs and combine the data tables into one spreadsheet.'
📦 Batch Processing Tip

If you have a folder of similar PDFs, Claude can process them all in one task: 'Read all PDFs in this folder and extract [specific data] from each into a single spreadsheet with one row per file.'

Working With Word Documents

Creating Professional Word Documents

The docx skill produces properly formatted Word documents with headings, tables, page numbers, headers, footers, and proper typography.

TaskWhat to Say
Report or analysis'Create a Word doc report on [topic] with headings, a summary table, and page numbers'
Meeting notes template'Create a Word doc meeting notes template for [meeting type]'
Project brief'Write a project brief for [project] as a Word doc'
Process documentation'Document the steps for [process] as a Word doc with numbered steps and callout boxes'

Editing Existing Word Documents

  • 'Update the dates in this document from 2024 to 2026'
  • 'Add a new section after the introduction covering [topic]'
  • 'Reformat this document to use proper heading styles and add a table of contents'
📝 Track Changes

Ask Claude to use tracked changes when editing: 'Edit this document with tracked changes so I can accept or reject each modification.'

Working With Spreadsheets

Analysing Data With Natural Language

  • 'Which products have the highest margin in this spreadsheet?'
  • 'Show me a month-by-month trend of the revenue column'
  • 'Find all rows where the status is pending and the date is more than 30 days ago'
  • 'Compare this year's sales to last year's by region'
🧹 Data Cleaning Prompt

'Clean this spreadsheet: standardise date formats to YYYY-MM-DD, remove duplicate rows, fill blank category cells with Unknown, and flag any rows where the total column does not match the sum of the line items.'

Document Pipelines

What Is a Document Pipeline?

A document pipeline is a sequence of automated steps that transforms raw inputs into polished outputs.

PipelineInputOutput
Invoice processingFolder of invoice PDFsSpreadsheet with all invoice data + summary totals
Research synthesisSet of research PDFs or URLsWord doc report with key findings
Meeting to actionMeeting notes text fileWord doc with agenda, decisions, and action items
Data to deckSpreadsheet of dataPresentation with charts and key insights per slide
Repo to documentationCode files from a repoWord doc with architecture overview and API reference

Designing a Document Pipeline Prompt

  1. State all input files and their locations
  2. Describe each transformation step
  3. Specify the output format and where to save it
  4. Include any quality standards (language level, length, formatting)
📋 Example Pipeline Prompt

'Read all five PDF reports in my outputs folder. For each, extract: the executive summary, key metrics, and recommendations. Combine all five into a single Word document with: a cover page, one section per report, and a final synthesis section that compares findings across all reports. Save as Synthesis_Report.docx.'

Practice

Exercises

Exercise 1

Exercise 1 — PDF Extraction (10 min)

Find any PDF on your computer. Ask Claude to extract the key information and save it as a structured Word doc or spreadsheet. Verify the output against the original.

Exercise 2

Exercise 2 — Create a Professional Document (15 min)

Ask Claude to create a Word doc summary of your marshmello project: what it does, its architecture, key dependencies, and open issues. Specify: professional formatting, a table of contents, and headings per section.

Exercise 3

Exercise 3 — Data Analysis (15 min)

Find any CSV or spreadsheet on your computer. Ask Claude to: clean the data, add a summary row, identify any outliers, and create a formatted Excel file.

Exercise 4

Exercise 4 — Build a Mini Pipeline (20 min)

Design a document pipeline relevant to your work. Write the pipeline prompt using the template from Module 5. Run it and evaluate the output quality.

Key Takeaways

  • Each document skill has a sweet spot — use the right one for the job.
  • PDF extraction unlocks data that is otherwise locked in static files.
  • The docx skill produces professional Word documents automatically.
  • Ask analytical questions about your spreadsheets in plain English.
  • Document pipelines are the highest-leverage way to use document skills — one prompt, many files, polished output.
